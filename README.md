# Chess Using Tkinter
Chess GUI made using tkinter and chess module in python.

![chess_tk](https://user-images.githubusercontent.com/110214573/236631024-477dc3f6-5baa-4844-81a7-e38e273624e1.png)

## Installation
To install tkinter and chess module:

```$ pip install tkinter```  
```$ pip install chess```
